Which of the following are purposes of the JdbcTemplate class?

A)
Reduces JDBC boilerplate code

B)​
Simplification of JDBC usage in Spring applications

C)​
Proper Exception Handling

D)​
Convenient methods for CRUD operations








































Answers 'A', 'B', 'C' and 'D'

Explanation

The JdbcTemplate provides all of the above benefits. 

https://docs.spring.io/spring/docs/5.0.7.RELEASE/javadoc-api/org/springframework/jdbc/core/JdbcTemplate.html
