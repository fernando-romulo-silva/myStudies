Which of these annotations can be used to insert a Mock of a Spring bean into the application context?

A)
@MockBean
​
B)
@InjectMockBean
​
C)
@InjectMocks
​
D)
@Mock







































Answers 'A'

Explanation

@MockBean can be used to inject a Mock instead of a Spring Bean into the Application Context. 

https://docs.spring.io/spring-boot/docs/current/api/org/springframework/boot/test/mock/mockito/MockBean.html
