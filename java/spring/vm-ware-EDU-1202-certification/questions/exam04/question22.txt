Which of the following exceptions, when thrown, will cause a Spring transaction rollback by default?

A)
None of the above.
​
B)
Error

C)​
RuntimeException

D)​
Checked exceptions








































Answers 'B' and 'C'

Only checked (business) exceptions will not cause transaction rollback. 

https://docs.spring.io/spring-framework/docs/current/javadoc-api/org/springframework/transaction/annotation/Transactional.html#rollbackFor--
