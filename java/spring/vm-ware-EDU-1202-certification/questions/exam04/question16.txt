In Spring MVC, a controller method argument may be annotated with _______.

A)
@PathVariable

B)​
@RequestParam

C)​
@RequestBody

D)​
@RequestMapping








































Answers 'A', 'B', 'C'

@RequestMapping may be applied only to Controller mapping methods. 

Annotating a handler method argument with one of the other 3 annotations will result in different parts of the request being mapped to the annotated arguments.

@RestController
public class RestFulController { 

	// http://localhost:8080/api/employees/111 
	@GetMapping("/api/employees/{id}")
	public String getEmployeesById(@PathVariable String id) {
	    return "ID: " + id;
	}
	
	// http://localhost:8080/api/foos?id=abc
	@GetMapping("/api/foos")
	public String getFoos(@RequestParam String id) {
	    return "ID: " + id;
	}	
	
	@PostMapping("/request")
	public ResponseEntity postController(@RequestBody LoginForm loginForm) {
		...
	}
}
