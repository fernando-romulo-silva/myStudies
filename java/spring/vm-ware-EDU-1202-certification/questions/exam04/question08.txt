Which of these is true about MVC?

A)
It gives a standard template for web application design.

B)​
It is a software design pattern

C)​
It allows reusing controllers for multiple views.

D)​
It is a framework for developing web applications.





































Answers 'A', 'B' and 'C'

MVC is a design pattern which provides set of guidelines for developing web applications.
