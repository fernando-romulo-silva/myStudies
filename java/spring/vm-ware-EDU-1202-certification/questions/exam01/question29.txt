Which class associates a request URL pattern with a list of filters in Spring Security?

A)
FilterChainProxy
​
B)
SecurityFilterChain

C)​
FilterChain
​
D)
DelegatingFilterProxy





































Answer 'B'

SecurityFilterChain is used by FilterChainProxy to determine which Spring Security Filters should be invoked for this request. 


https://docs.spring.io/spring-security/site/docs/current/reference/html5/#servlet-securityfilterchain
