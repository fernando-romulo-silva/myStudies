The annotation __________ enables Spring Boot auto-configuration

A)
@AutoConfiguration
​
B)
@SpringConfiguration
​
C)
@Configuration
​
D)
@EnableAutoConfiguration





































Answer 'D'

@EnableAutoConfiguration is used to enable Spring Boot’s auto-configuration mechanism. 

@Configuration is an annotation used on classes which contain bean definition methods.
