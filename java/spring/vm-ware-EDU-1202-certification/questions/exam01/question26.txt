On which of these database system isolation levels can phantom reads occur?

A)
READ_COMMITED
​
B)
SERIALIZABLE
​
C)
REPEATABLE_READ
​
D)
READ_UNCOMMITED





































Answer 'A', 'C' and  'D'

Phantom reads can occur in all database system isolation levels except for SERIALIZABLE.
