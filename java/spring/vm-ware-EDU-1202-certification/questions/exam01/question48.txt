Which of the following bean scopes can only be used in the context of a web-aware Spring Application Context?

A)
web
​
B)
session

C)​
application

D)​
request





































Answer 'B'

The following bean scopes are only valid in the context of a web-aware Spring ApplicationContext:
 
* session

* request

* application

* websocket

There is no 'web' bean scope out of the box. 

https://docs.spring.io/spring-framework/docs/current/spring-framework-reference/core.html#beans-factory-scopes
