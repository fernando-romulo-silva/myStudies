Which of the following HTTP verbs can be used to create a new entity, but not modify an existing one?
​
​A)
GET
​
B)
POST

C)​
DELETE
​
D)
PUT
   








































Answer 'B'

Explanation

Function of POST: Create a member resource in the collection resource using the instructions in the request body. 

The URI of the created member resource is automatically assigned and returned in the response Location header field.
