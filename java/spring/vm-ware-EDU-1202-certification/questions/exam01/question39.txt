In order to define a bean, one can create a class annotated with ________ and add a method annotated with _________ to it.

A)
@Component, @ManagedBean
​
B)
@Service, @BeanDefinition
​
C)
@Configuration, @Inject
​
D)
@Configuration, @Bean





































Answer 'D'

The Annotation @BeanDefinition does not exist. 
The annotation @ManagedBean is not part of Spring (it is a JEE class annotation similar to @Component). 

https://docs.spring.io/spring/docs/current/spring-framework-reference/core.html#beans-java

@Configuration
public class ImageConfig {

    @Bean
    public Validator validator() {
		final var factory = Validation.buildDefaultValidatorFactory();
		return factory.getValidator();
    }
}
