Which of the following AOP advice types can choose to prevent execution of the advised method, and instead return another value?

A)
@Around

B)​
@AfterReturning
​
C)
@Before
​
D)
@AfterThrowing





































Answer 'A'

@Before advice can also prevent execution of the advised method, however only if an exception is thrown.

@Around can prevent the execution of the advised method and change the return value. 

https://docs.spring.io/spring-framework/docs/current/spring-framework-reference/core.html#aop-ataspectj-around-advice
