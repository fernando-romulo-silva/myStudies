What effect does setting the attribute "readOnly" on the @Transactional annotation to true have?

A)
Does not allow write operations
​
B)
Nothing. It is there only for documentation.
​
C)
May optimize query performance

D)​
Sets the lock mode to READ





































Answer 'C'

Only some databases reject the INSERT and UPDATE statements inside a read only transaction.
