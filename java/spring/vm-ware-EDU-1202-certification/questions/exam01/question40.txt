What do SpEL expressions starting with @ reference?

A)
JVM Properties
​
B)
Literal Values
​
C)
Spring Beans

D)​
Properties in the application environment





































Answer 'C'

A Spring bean is referenced using its name prefixed with @ in SpEL. 

Chains of property references can be accessed using the period character. 


Object bean = parser.parseExpression("@foo").getValue(context);


https://docs.spring.io/spring-framework/docs/5.0.3.RELEASE/javadoc-api/org/springframework/expression/spel/standard/SpelExpression.html
