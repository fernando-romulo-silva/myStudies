Which of the following are true about Spring MVC Controllers?

A)
When using annotations, specific class or interface inheritance is not required.

B)​
@Controller is a stereotype annotation.

C)​
Endpoint paths mapped by a controller must all have the same prefix.
​
D)
Controllers implemented using annotations do not have direct dependencies on Portlet or Servlet APIs.





































Answer 'A', 'B' and 'D'

Endpoint Paths do not necessarily need to have the same path prefix. 

@Controller is a stereotype annotation because it is meta annotated with @Component. 

Before annotation based controller declaration, controllers needed to extend certain classes, such as MultiActionController.


@Controller
public class HelloWorldController {

    @RequestMapping("/helloWorld")
