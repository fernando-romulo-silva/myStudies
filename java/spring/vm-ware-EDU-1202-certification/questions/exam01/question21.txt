Which exception can potentially be thrown when calling the JdbcTemplate.query() method?

A)
SQLException
​
B)
JdbcException
​
C)
QueryException

D)​
DataAccessException





































Answer 'D'

DataAccessException is the root exception thrown by all Data Access methods. 

Any SQLExceptions will be suppressed by a DataAccessException. 

https://docs.spring.io/spring/docs/current/javadoc-api/org/springframework/jdbc/core/JdbcTemplate.html#query-org.springframework.jdbc.core.PreparedStatementCreator-org.springframework.jdbc.core.RowMapper-
