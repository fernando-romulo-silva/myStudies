What does the abbreviation MVC stand for?
​
​A)
Model View Compiler
​
B)
Multi Variable Control
​
C)
Model-View-Controller
​
D)
Multi Variable Compiler








































Answer 'C'

Explanation

MVC stands for Model-View-Controller. 
It is a programming paradigm which provides separation of concern, which makes it easy to develop and maintain a web application.
