_________ is linking aspects with other application types or objects to create an advised object

A)
Pointcut

B)​
Weaving

C)​
Advice
​
D)
Aspect





































Answer 'B'

Weaving: linking aspects with other application types or objects to create an advised object. 

This can be done at compile time (using the AspectJ compiler, for example), 
load time, or at runtime. Spring AOP, like other pure Java AOP frameworks, performs weaving at runtime.  


https://docs.spring.io/spring-security/site/docs/current/reference/html5/#servlet-securityfilterchain
