Which of the following is a feature of Spring Boot Actuator?
​
A)
Monitoring
​
B)
Management

C)​
Circuit-Breaker
​
D)
Metrics
   








































Answer 'A', 'B' and 'D'

Explanation

Monitoring, Metrics and Management are provided by Spring Boot Actuator. 
Ribbon and Hystrix are the tools provided by Spring as circuit breakers.
