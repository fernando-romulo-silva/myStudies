In Spring Security, how can the usage of @Secured annotations be enabled?
​
A)
@EnableGlobalMethodSecurity(prePostEnabled = true)
​
B)
@EnableGlobalMethodSecurity(jsr250Enabled = true)

C)​
They are enabled by default.
​
D)
@EnableGlobalMethodSecurity(securedEnabled = true)








































Answer 'D'

Explanation

@EnableGlobalMethodSecurity provides AOP security on methods. 

Some of the annotations that it provides are PreAuthorize, PostAuthorize. 

It also has support for JSR-250.

The attribute securedEnabled specifies whether the @Secured annotation can be used in the application. 

https://docs.spring.io/spring-security/site/docs/current/reference/html5/#enableglobalmethodsecurity
