______ means that none of the actions taken in a transaction violate any integrity constraints.
  ​
A)
Durability
​
B)
Consistency
​
C)
Atomicity
​
D)
Isolation
   








































Answer 'B'

Transaction follows ACID properties, where C stands for Consistency. 

According to it, transactions should be consistent. 

There should be no loss of data after a transaction.
