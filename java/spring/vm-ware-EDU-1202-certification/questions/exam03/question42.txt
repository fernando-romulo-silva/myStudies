Which of the following are advantages of running web application tests on a random port?
  ​
A)
Security
​
B)
Integration testing
​
C)
Parallelization

D)​
Debugging capabilities








































Answer 'C'

The advantage when running web application tests on a random port is that it will not clash 
with other instances of the application (such as a running application or other tests)

https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/#boot-features-testing-spring-boot-applications-testing-with-running-server
