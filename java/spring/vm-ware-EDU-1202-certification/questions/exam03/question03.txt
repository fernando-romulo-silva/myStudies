Which is the correct way of selecting the "dev" Spring profile using command line arguments?
  ​
A)
-spring.profiles.active=development
​
B)
spring.profiles=dev
​
C)
--spring.profiles.active=dev
   








































Answers 'C'

Application properties can be changed in command line arguments by prefixing the property names with "--". 

$ java -Dspring.profiles.active=dev -jar yourApplication.jar 

https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/#boot-features-profiles
