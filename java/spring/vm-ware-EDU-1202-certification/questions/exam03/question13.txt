In Spring Security, what is the name of the class holding the information regarding 
high level user permissions?
  ​
A)
PrincipalAuthority

B)​
Authority
​
C)
UserDetails
​
D)
GrantedAuthority
   








































Answers 'D'

A granted authority is an authority that is granted to the principal on the Authentication (i.e. roles, scopes, etc.) 

https://docs.spring.io/spring-security/site/docs/current/reference/html5/#servlet-authentication

private Collection<? extends GrantedAuthority> getAuthorities(Collection<Role> roles) {
 
    List<GrantedAuthority> authorities = new ArrayList<>();
    
    for (Role role: roles) {
        authorities.add(new SimpleGrantedAuthority(role.getName()));
        role.getPrivileges().stream()
            .map(p -> new SimpleGrantedAuthority(p.getName()))
            .forEach(authorities::add);
    }
    
    return authorities;
}
