Which of the following are ways to change the behavior of a Spring Boot application in a certain profile?
  ​
A)
@PropertySource annotation
​
B)
application-{profile}.properties
​
C)
@Profile annotation

D)​
application-{profile}.yml








































Answers 'B', 'C' and 'D'

The following ways of changing the behaviour based on profiles exist: 

@Profile annotations in java code, 

application-{profile}.properties 

and its YAML variant. 

https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/#boot-features-external-config
