Which of the following statements are true regarding the Spring Application Context?

A)​
It resolves external configuration properties.

B)​
It is the front servlet of the application.
​
C)
A running application has only one.
​
D)
It instantiates Spring Beans





































Answers 'A' and 'D'

While the beans package provides basic functionality for managing and manipulating beans, including in a programmatic way

The Front Servlet of the application is the DispatcherServlet. 

It is possible for applications to have multiple applications context, arranged in a parent-child hierarachy.
