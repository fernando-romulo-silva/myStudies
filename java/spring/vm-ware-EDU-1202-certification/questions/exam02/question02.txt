Which of the following are endpoints provided by Spring Boot Actuator?
  ​
A)
mappings

B)​
users
​
C)
liquibase

D)​
flyway








































Answer 'A', 'B', and 'D'

mappings: Lists the @RequestMapping paths of the application. 

flyway: Lists applied Flyway database migrations. 

liquibase: Lists applied Liquibase database migrations. 

Spring Boot Actuator does not have an endpoint called users.
