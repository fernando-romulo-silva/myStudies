Which of the following are characteristics of Spring Security?

A)​
It is a cross cutting concern
​
B)
It is based on standard Servlet Filters
​
C)
All of the above.

D)​
It provides options for both Authentication and Authorization





































Answer 'C'

Spring Security is a cross cutting concern, so it is implemented using Spring AOP. 

It provides multiple options for authorization as well as authentication. 

It uses standard Java Servlet Filters. 

https://docs.spring.io/spring-security/site/docs/current/reference/html5/#servlet-applications
