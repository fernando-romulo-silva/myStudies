Which annotation can be used to have the request body read and deserialized?
  ​
A)​
@RequestDeserialize
​
B)
@RequestBody

C)​
@RequestMapping
​
D)
@JsonDeserialize






































Answer 'B'

@JsonDeserialize is not an annotation in Spring. 

@RequestMapping is used to configure controllers and their handler methods. 

@RequestDeserialize does not exist. 

@RequestBody is used for access to the HTTP request body. Body content is converted to the object
