What is the name of the default property configuration file in Spring Boot?
  ​
A)
configuration.properties
​
B)
config.txt
​
C)
settings.properties
​
D)
application.properties







































Answer 'D'

The default configuration file is application.properties (or application.yml if YAML is used). 

https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/#boot-features-external-config
