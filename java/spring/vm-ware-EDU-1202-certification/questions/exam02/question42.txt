@SpringBootTest is meta annotated with ________.

A)​
@ExtendWith(SpringExtension.class)

B)​
@RunWith(SpringRunner.class)
​
C)
@RunWith(SpringExtension.class)

D)​
@ExtendWith(SpringRunner.class)





































Answer 'A'

@SpringBootTest is meta annotated with @ExtendWith(SpringExtension.class), because it uses JUnit 5. 

@RunWith(SpringRunner.class) is the annotation used with JUnit 4.
