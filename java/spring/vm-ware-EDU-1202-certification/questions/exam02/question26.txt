Which of the following are possible build output types when using Spring Boot?
  ​
A)​
All of the above.

B)​
Non-executable JAR
​
C)
WAR
​
D)
Fat JAR






































Answer 'A'

Spring Boot offers options of building the application as an executable (fat) JAR, non-executable JAR or WAR. 

https://docs.spring.io/spring-boot/docs/2.3.4.RELEASE/reference/htmlsingle/#howto-build
