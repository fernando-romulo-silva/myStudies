_______ is used to map URL Query String Parameters to handler method arguments,
  ​
A)
@SessionParam
​
B)
@PathVariable
​
C)
None of the above.
​
D)
@RequestParam







































Answer 'D'

When an annotation @RequestParam is used before a variable of a controller method argument, 
the value of query parameter which is part of URL gets copied into the same variable of the controller method.

GET: http://localhost:8080/api/foos?id=abc

@GetMapping("/api/foos")
@ResponseBody
public String getFoos(@RequestParam String id) {
    return "ID: " + id;
}
