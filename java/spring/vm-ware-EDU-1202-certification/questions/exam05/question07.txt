A Class implementing the interface ________ have additional methods to ease paginated access to entitites.
  ​
A)
PaginatedRepository

B)    ​
PagingAndSortingRepository

C)    ​
JpaRepository

D)
CrudRepository







































Answer 'B'

The PagingAndSortingRepository abstraction adds additional methods to ease paginated access to entities.


public interface ProductRepository extends PagingAndSortingRepository<Product, Integer> {

    List<Product> findAllByPrice(double price, Pageable pageable);
}
