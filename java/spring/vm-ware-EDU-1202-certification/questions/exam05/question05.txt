What is a JAR file with all needed dependencies needed to run the application called?
  ​
A)
application JAR
​
B)
dependency JAR
​
C)
full JAR
​
D)
fat JAR







































Answer 'D'

Executable jars (sometimes called “fat jars”) are archives containing your compiled classes along with all of the jar dependencies that your code needs to run. 

https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/#getting-started-first-application-executable-jar
