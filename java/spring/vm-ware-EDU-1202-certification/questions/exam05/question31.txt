The annotation ______ instructs the IoC container to prepare the specified bean before initializing the annotated bean definition.

A)
@Configuration

B)
@Lazy

C)
@Bean

D)
@DependsOn











































Answer 'D' 

When @DependsOn is used, the IoC Container makes sure that the dependency is available before the annotated bean is instantiated.

 https://docs.spring.io/spring-framework/docs/current/javadoc-api/org/springframework/context/annotation/DependsOn.html
