Which of the following are true about Spring Framework's Test module?
  ​
A)
JUnit is supported

B)​
It supports Mockito Framework
​
C)
AspectJ is included as a dependency
​
D)
It supports EasyMock








































Answers 'A', 'B' and 'D'

Spring Framework's Test module offers support for JUnit, Mockito and EasyMock.  
AspectJ is an AOP dependency which is not included or required for the Spring Test Module.  

http://docs.spring.io/spring-framework/docs/4.0.9.RELEASE/spring-framework-reference/html/overview.html#overview-testing
