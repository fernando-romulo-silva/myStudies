Which of these is ordering operator which can be used in custom finder method declaration in Spring Data JPA Repository Interface?
  ​
A)
​Asc
​
B)
Descending
​
C)
Ascending
​
D)
Desc








































Answers 'A' and 'D'

The correct ordering operators are Asc and Desc. 

public interface StudentDAO extends JpaRepository<StudentEntity, Integer> {
    
    public List<StudentEntity> findAllByOrderByIdAsc();
    
    public List<StudentEntity> findAllByOrderByIdDesc();
}
