Which functions are possible only when using interfaces as spring beans?

A)
JDK Dynamic Proxying

B)
Exception Handling

C)
Test Application Context

D)
Dependency Injection










































Answer 'A' 

JDK Dynamic Proxies work by proxying interfaces. 
If the classes do not have interfaces, then only CGLIB Proxies can be used (as these generate subclasses). 

https://docs.spring.io/spring/docs/current/spring-framework-reference/core.html#aop-introduction-proxies

https://docs.spring.io/spring/docs/current/spring-framework-reference/core.html#aop-understanding-aop-proxies
