Which of the following bean scopes exist?

A)
All of the above

B)
webapp

C)
prototype

D)
local

E)
request












































Answers 'C' and 'E'

There are six scopes for beans in Spring framework namely: 

* singleton (default)

* prototype

* application

* request (web)

* session (web)

* websocket (web) 


Local and Webapp Bean scopes do not exist. 

https://docs.spring.io/spring/docs/current/spring-framework-reference/core.html#beans-factory-scopes
