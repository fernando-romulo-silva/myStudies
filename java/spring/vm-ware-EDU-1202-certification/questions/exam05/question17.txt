Which of these is comparison operator which can be used in a finder method declaration 
in Spring Data JPA Repository Interface, which will also produce a valid database query?
  â€‹
A)
Like

B)
Is

C)
GreaterThan

D)

Answers 'A', 'B', 'C', 'D'

All of these are valid comparison operators. "Is" is equivalent to "Equals". 


List<Person> findByFirstnameLike(final String name);

List<Person> findByFirstnameIs(final String firstName);

List<Person> findByAgeGreaterThan(final int age);

List<Person> findByStartDateBetween(final Date startDate)
