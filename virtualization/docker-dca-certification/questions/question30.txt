How can we filter running containers based on a specific image?


a) There is no option for this. We use the Linux grep command to filter specific base images for containers.

b) We will use the ancestor key to list all the running containers using a specific image.

c) We will use the image key to list all the running containers using a specific image.

d) None of the preceding sentences are right.








































Answer 'b' because of

$ docker ps -q  --filter ancestor=docker-mariadb:1.0.1
