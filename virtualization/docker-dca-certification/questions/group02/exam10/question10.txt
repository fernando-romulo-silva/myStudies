What is the docker run option to publish a port so that an application is accesible externally?

A)
docker run --open-port

B)
docker run --publish

C)
docker run --expose

D)
docker run --publish-port 








































Answer B

The param --publish or -p publish a container's port(s) to the host

$ docker run --name test-mysql --detach --publish 3306:3306 mysql:latest
