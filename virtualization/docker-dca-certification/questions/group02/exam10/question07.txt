What is the docker command to add or update a published port?

A)
docker service set --add-port

B)
docker service modify --add-update-port

C)
docker service update --publish-port

D)
docker service update --publish-add





































Answer D

You can change almost everything about an existing service using the docker service update command. 
When you update a service, Docker stops its containers and restarts them with the new configuration.

$ docker service update --publish-add
