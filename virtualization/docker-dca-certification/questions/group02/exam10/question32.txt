You have deployed a service to swarm. 
Which command uses the Docker CLI to set the number of tasks of the services to 5? (choose 2)

A)
'docker replica update =5'

B)
'docker service replicas =5'
C)
'docker update service =5'

D) 
'docker service update --replicas=5 '

E)
'docker service scale = 5''


































Answers D and E

The command 'scale' Scale one or multiple replicated services.

$ docker service scale frontend=50

You can also scale a service using the docker service update command. 
The following commands are equivalent:

$ docker service update --replicas=50 frontend
