When using the Docker client to push an image to a registry, 
what environment variable is used to instruct the client to perform signing of the image?

A)
NOTARY_ENABLE=1

B)
DOCKER_PUSH_SIGN=1

C)
DOCKER_IMAGE_SIGN=1

D)
DOCKER_CONTENT_TRUST=1 




































Answer D

For the Docker client to know to use trusted server (environment), 
you will need to set an environment variable pointing to it:

$ export DOCKER_CONTENT_TRUST=1
