Which of the following is the default mount point for a decrypted secret in a Linux container?

A)
/run/secrets/

B)
/var/lib/docker/secrets/

C)
/run/internal/secrets/

D)
/var/lib/docker/internal/secrets/





































Answer A

The location of the mount point within the container defaults to /run/secrets/<secret_name> in Linux containers, 
or C:\ProgramData\Docker\secrets in Windows containers. 

You can also specify a custom location.
