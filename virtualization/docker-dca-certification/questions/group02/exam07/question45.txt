You want to initialise scan of your repository for vulnerabilities. 
But you are unable to do so. 
What could be the reason?

A)
You are using Docker Community Edition.

B)
Security Scanning in not enabled in DTR.

C)
You do not have the authority to download the security scanning licence for DTR.

D)
All of the above.  


































 
Answer D

To run vulnerability scanning on your Docker images, you must meet the following requirements:
 
* DTR license that includes Docker Security Scanning

* Your Docker ID can access and download this license from Docker Hub.
