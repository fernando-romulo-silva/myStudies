Which docker command can be used to find all the ports mapped? 
(select all that apply).

A)
docker ps

B)
docker inspect

C)
docker network ls

D)
docker port 




































Answers A, B and D

Docker port command only shows the port published externally.

docker ps:

$ docker ps
CONTAINER ID   IMAGE       COMMAND                  CREATED        STATUS         PORTS                                                  NAMES
7a38e315e870   mysql:5.7   "docker-entrypoint.sâ€¦"   6 months ago   Up 8 minutes   0.0.0.0:3306->3306/tcp, :::3306->3306/tcp, 33060/tcp   murachs-mysql

docker port test:

$ docker port murachs-mysql
3306/tcp -> 0.0.0.0:3306
3306/tcp -> :::3306


docker inspect: 

$ docker inspect --format='{{(index (index .NetworkSettings.Ports "3306/tcp") 0).HostPort}}' murachs-mysql 
3306
