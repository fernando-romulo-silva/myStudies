Which command can you use to display the layers of a docker image?

A)
docker image history

B)
docker layers

C)
docker image layers

D)
docker image info





































Answer A

You can examine all the layers in an image by using the docker image history command. 
It will display the following:

路 Abbreviated layer ID
路 Age of the layer
路 Initial command of the creating container
路 Total file size of that layer
