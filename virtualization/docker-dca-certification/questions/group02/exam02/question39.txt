Which of the following docker component can be described as “a runtime instance of an image”?

A)
Docker File

B)
Node

C)
Container

D)
Image





































Answer C

A container is a runtime instance of a docker image.

A Docker container consists of
· A Docker image
· An execution environment
· A standard set of instructions

The concept is borrowed from Shipping Containers, which define a standard to ship goods globally. 
Docker defines a standard to ship software.
