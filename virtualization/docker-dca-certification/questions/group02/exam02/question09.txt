A ____ is an instance of the Docker engine that participates in the swarm.

A) 
Node

B) 
Service

C)
Container

D)
Server





































Answer A

A node is an instance of the Docker engine participating in the swarm. You can also think of this as a Docker node.

You can run one or more nodes on a single physical computer or cloud server, 
but production swarm deployments typically include Docker nodes distributed across multiple physical and cloud machines.
