Recent problems with the service called ‘my_api’ in your cluster have caused your development team to request its removal 
from the cluster while they address the problems.

Which command do you need to execute to remove the service?

A)
docker swarm remove my_api

B)
docker rm my_api

C)
docker service rm my_api

D)
docker swarm --remove=my_api


































Answer C

To remove one or more services from the swarm use:
docker service rm SERVICE [SERVICE…]
Note: This is a cluster management command, and must be executed on a swarm manager node.
