When querying the configuration of a running container called ‘webApp’, 
which command would display the container’s IP Address?

A)
docker run inspect webApp | grep IP

B)
None of the answers is correct

C)
docker inspect webApp | grep IPAddress

D)
docker webApp inspect | grep IP




































Answer C

To return low-level information on Docker objects use:
docker inspect [OPTIONS] NAME|ID [NAME|ID…]

$ docker inspect myweb | grep IPAddress
