What are the hardware and software requirements to install Docker Trusted Registry (DTR)? 
(select all that apply)

A)
Make sure the port 8080 and 8443 are open on the node

B)
All nodes must be a worker node managed by Universal Control Plan

C)
All nodes must have fixed hostname

D)
DTR can be installed on-premises or on a cloud provider




































Answers B, C and D

You can install DTR on-premises or on a cloud provider. 

To install DTR, all nodes must:

· Be a worker node managed by UCP (Universal Control Plane). 
  DTR replicas are installed one on each UCP worker node.
  
· Have a fixed hostname.
