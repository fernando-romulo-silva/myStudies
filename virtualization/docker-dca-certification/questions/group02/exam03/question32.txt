Which is the recommended way to configure the daemon flags and environment variables for your Docker daemon in a platform-independent way?

A)
Using 'docker config' to set the configuration options.

B)
Set the configuration options using the ENV variable

C)
Set the configuration DOCKER_OPTS in '/etc/default/docker'

D)
Set the configuration options in '/etc/docker/daemon.json'





































Answer D

There are a number of ways to configure the daemon flags and environment variables for your Docker daemon. 

The recommended way is to use the platform-independent daemon.json
