Which of the following is NOT a docker native network driver?

A)
host

B)
bridge

C)
load balancer

D)
macvlan





































Answer C

Supported network drivers for Docker are: bridge, host, overlay, macvlan, and none. 
Also, Docker supports third-party network plugins available on Docker Hub or from third party vendors. 
For more information, Please follow the link: https://docs.docker.com/network/
