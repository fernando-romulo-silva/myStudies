Which of the following Linux technologies is responsible for the isolated nature of containers?

A)
Isolated Networks

B) 
namespace

C)
core isolation

D)
cgroups





































Answer B

When a container starts running, each process of a container runs in a separate namespace. 
These namespaces provide a layer of isolation to the respective processes in a container and limit their functionality to that particular namespace. 
For more information, please follow the link: https://docs.docker.com/engine/docker-overview/
