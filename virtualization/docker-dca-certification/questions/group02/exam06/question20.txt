Which of the following provisions of Docker supports modifications on storage drivers?

A)
Docker for Linux Distributions

B)
Docker Desktop for Mac

C)
Docker Desktop for Windows

D)
All of the above





































Answer A

Modifying the storage-driver is not supported on Docker Desktop for Mac and Docker Desktop for Windows, 
and only the default storage driver can be used.
 
The comparison table below is also not applicable for Rootless mode.


Ubuntu 	overlay2 	overlay¹, devicemapper², aufs³, zfs, vfs
