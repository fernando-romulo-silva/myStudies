Image scan is a feature of Docker trusted Repository DTR, which is part of Docker Entreprise Edition.

To use the image scanning feature, make sure that you or your organization has purchased a DTR license that includes Docker Security Scanning, 
and that your Docker ID can access and download this license from the Docker Hub.
