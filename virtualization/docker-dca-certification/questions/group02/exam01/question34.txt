Which of the following commands can you use to pretty-print the containers 
IDs, names, and statuses in tabular form?

A)
docker ps --format "table {{.ID}}\t {{.Names}}\t {{.Status}}"

B)
docker ps -format table {?{.ID}} {?{.Names}} {?{.Status}}

C)
docker ps --format {?{.ID}} {?{.Names}} {?{.Status}}

D)
docker ps --format table {?{.ID}} {?{.Names}} {?{.Status}} 




































Answer A

The formatting option –format pretty-prints container output using a Go template.
When using the –format option, the ps command will either output the data exactly as the template declares or, 
when using the table directive, includes column headers as well.

The following example uses a template without headers and outputs the ID, name and status entries 
separated by a colon (:) for all running containers.

Use “\t” for extra space between columns.

$ docker ps --format "table {{.ID}}\t {{.Names}}\t {{.Status}}"

CONTAINER ID    NAMES                        STATUS
866ac42175df    image-converter-service-1    Up 25 seconds
