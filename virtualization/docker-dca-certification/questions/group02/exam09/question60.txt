What is the docker command to display detailed information on one or more networks?

A)
docker inspect network

B)
docker network status

C)
docker network display

D)
docker network inspect 





































Answer D

docker network inspect returns information about one or more networks. 

By default, this command renders all results in a JSON object.

$ docker network inspect test-net -f "{{json .Containers }}"

or

$ docker inspect container1 -f "{{json .NetworkSettings.Networks }}"
