Which of the following is a pseudo-file containing the CPU usage accumulated by the processes of a container?

A)
cpuusage.stat

B)
cpuacct.stat

C)
cpushares.stat

D)
cpus.stat





































Answer B

For each container, a pseudo-file cpuacct.stat contains the CPU usage accumulated by the processes of the container, broken down into user and system time.
