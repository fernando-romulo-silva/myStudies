

What does the docker swarm --force-new-cluster command do?

a) It is used to recover a cluster in failure situations. It will set all managers as workers, leaving just one manager node.

b) This command will destroy the cluster. It is used to remove the entire cluster.

c) --force-new-cluster should be used to stop all services deployed on worker nodes.

d) An application's services will not be impacted by this command. This command just affects the control plane.







































Answer(s) 'a'

It's force create a new cluster from current state
This flag forces an existing node that was part of a quorum that was lost to restart as a single node Manager without losing its data.
