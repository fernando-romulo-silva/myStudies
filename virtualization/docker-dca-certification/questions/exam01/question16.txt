
Which of these sentences are true about signing images?

a) Image signing ensures image ownership.

b) Images can be signed using docker trust sign <IMAGE>.

c) All images will be signed if we set the Docker client to use Docker Content Trust on every command.

d) Image signing is based on the following keys: the owner key, the repository key, the snapshot key, and the timestamp.








































Answer 'a', 'b', and 'c'

a - To sign a Docker Image you will need a delegation key pair, with that Image signing ensures image ownership.

// load the key

$ docker trust key load key.pem --name jeff


b - Trust the image

$ docker trust sign example/trust-demo:v2


c - Since the commands execute only in trust way
