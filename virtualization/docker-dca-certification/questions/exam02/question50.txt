Which command is used to create an overlay network that can be used with swarm services?

A)
docker make overlay network 'network name'
​
B)
docker network create -d overlay 'network name'
​
C)
docker-network create 'network name'
​
D)
docker network 'network name' create -d overlay





































B is correct

To create an overlay network for use with swarm services, use a command like the following:

$ docker network create -d (--driver) overlay my-overlay
