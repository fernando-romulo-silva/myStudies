Which of the following statement is false about the public registry?


A) The public registry helps in creating public Dockers.
​
B) The public registry also called a docker hub.
​
C) Docker hub permits you for storing images privately.
​
D) There is the availability of millions of images in the Docker hub.








































A is the correct option

Public registry also called a docker hub, Docker hub permits you for storing images privately, 

There is the availability of millions of images in docker hub are true about docker. 

But the first statement is incorrect because docker permits for creating private image storage.
