From the below-mentioned option, what are the correct option which helps in 
easily letting the communication happen between docker demons and docker client?

​
A) Rest API
​
B) Socket.IO
​
C) PCP
​
D) All of the above







































The correct option is D

For creating communication between docker client and docker demons the combination 
of three things is required, which is as follows, A. Rest API, Socket.IO, PCP. 

Hence this all three are helpful for the above-mentioned communication.
