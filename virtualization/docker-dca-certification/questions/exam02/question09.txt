What are the requirements to install Docker Trusted Registry (DTR)? (Select all that apply)

A) Make sure the port 8080 and 8443 are open on the node
​
B) All nodes must be a worker node managed by Universal Control Plan
​
C) DTR can be installed on-premises or on a cloud provider
​
D) All nodes must have fixed hostname









































The correct option is B e C

You can install DTR on-premises or on a cloud provider. To install DTR, all nodes must:

Be a worker node managed by UCP (Universal Control Plane). See Compatibility Matrix for version compatibility.
Have a fixed hostname.
