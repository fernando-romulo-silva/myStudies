When you create a new container, you add a new ______ layer on top of the underlying layers. 
This layer is often called the “container layer”)

A)
read only

B)
writable

C)
filesystem

D)
volume




































Answer  B

When a container is created docker adds a new writable layer on top of the underlying layers. 
This layer is called container layer. 
This layer is deleted when the container is deleted.
