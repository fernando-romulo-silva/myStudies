"docker service create nginx" is a command that creates a service in docker swarm. 
Which of the following statements is NOT TRUE?

A)
This command can only be run from a manager node

B)
Your service will not be able to serve inbound connections

C)
A replica will run either from a manager or from a worker node

D)
This service will only run from a worker.




































Answer  D

A Is true because we can create a service from manager node only.

B No inbound connection will be served because no published port was specified. You have to expose it.

C Both workers and managers process workloads.
