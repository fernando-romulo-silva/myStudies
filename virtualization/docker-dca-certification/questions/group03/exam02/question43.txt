Docker Content Trust provides the ability to use __________ for verifying integrity and the publisher of all the data received from a registry over any channel

A)
end to end encryption

B)
digital signatures

C)
symmetric key encryption

D)
a container





































The correct answer is Option B.

Option B is correct because digital signatures can be used to verify integrity of both the content as well as the publishers using public private key encryption model

Option A is incorrect because end to end encryption is used for transmitting encrypted data between two nodes

Option C is incorrect because symmetric key encryption is used for transmitting encrypted data

Option D is incorrect. Docker content trust doesnâ€™t provide a container for this purpose
