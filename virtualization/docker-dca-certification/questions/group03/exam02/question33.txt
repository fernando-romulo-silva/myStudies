You are creating a new Docker Swarm. This docker Swarm needs to use a subnet range of 172.16.2.0/24 in 
its overlay netowrk to avoid collision with other networks.
 
Which command below would enforce the usage of this subnet range?

A)
docker swarm init --addr-pool 172.16.2.0/24

B)
docker swarm init --default-addr-pool 172.16.2.0/24

C)
docker swarm init --address-pool 172.16.2.0/24

D)
docker swarm init --address-range 172.16.2.0/24


































Answer  B

As per docker documentation “--default-addr-pool” is the the flag used to specify address pool while creating docker swarm.
