Which one is a valid command to run a global service in docker swarm?

A)
docker service create --mode global

B)
docker swarm create --mode global

C)
docker service create --global

D)
docker swarm create --global

Answer A

For global services, the service runs on every node that meets the placement constraint and any resource requirements.

$ docker service create \
  --name my-nginx \
  --mode global \
  --constraint node.labels.region==east \
  --constraint node.labels.type!=devel \
  nginx
