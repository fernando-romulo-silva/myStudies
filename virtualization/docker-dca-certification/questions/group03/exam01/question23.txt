Which of the following commands can be used to check Entrypoint and CMD of a given docker image?

A)
docker image inspect

B)
docker image ls

C)
docker info image

D)
docker image info





































Answer A

a) This option is correct

b)  “docker image ls” is the command used for listing all available images in docker host

c) docker info image is not a valid command. “docker info” gives information about docker host and swarm state.

d) “docker image info” is not a valid command.
