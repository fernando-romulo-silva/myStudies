Bob did a fresh installation of docker on his new linux server. 
He hasn’t tinkered with anything and has just installed docker packages from official repository. 
He runs a new container with command “docker run -d nginx” , 
which logging driver will this container use?

A)
syslog

B)
logentries

C)
json-file

D)
journald


































Answer C

json-file is the default logging driver in docker
