Bob has configured his docker daemon to use syslog log driver by default, 
how can he run a container which uses json-file log driver?

A)
By using “--log-opt json-file” along with docker run

B)
By using “--log-driver json-file” along with docker run

C)
By using “--logger json-file” along with docker run

D)
Not possible




































The correct answer is Option B.

$  docker run \
      --log-driver json-file --log-opt max-size=10m \
      alpine echo hello world
