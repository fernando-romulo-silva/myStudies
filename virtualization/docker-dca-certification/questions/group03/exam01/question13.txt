What is docker command to create swarm?

A)
docker swarm initialize

B)
docker swarm start

C)
docker swarm join

D)
docker swarm init



Answer D

"init” subcommand is used for creating new docker swarm cluster. 
The node on which this command runs by default becomes manager node of the swarm.
This command works with the Swarm orchestrator.

$ docker swarm init [OPTIONS]


“initialize” is not a valid docker swarm subcommand

"start” is not a valid docker swarm subcommand

“join” is docker swarm subcommand used for joining an existing docker swarm cluster
