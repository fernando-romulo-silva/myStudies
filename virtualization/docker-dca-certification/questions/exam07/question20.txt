
What is the purpose of Docker Content Trust?

a) Signing and verification of image tags

b) Indicating an image on Docker Hub is an official image
â€‹
c) Enabling mutual TLS between the Docker client and server

d) Docker registry TLS verification and encryption








































Answer 'a'

Docker Content Trust (DCT) provides the ability to use digital signatures for data sent to and received from remote Docker registries. 
These signatures allow client-side or runtime verification of the integrity and publisher of specific image tags.
