A server is running low on disk space. 
What command can be used to check the disk usage of images, containers, and volumes for Docker engine?

a)​ 'docker system df'

b) 'docker system prune'
​
c) 'docker system ps'

d) 'docker system free'













































'A'

The docker system df command displays information regarding the amount of disk space used by the docker daemon.

$ docker system df
