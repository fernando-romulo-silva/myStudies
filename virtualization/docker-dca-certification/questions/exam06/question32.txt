What is the command that need to be executed to sign an image before pushing it to repository?

a)​ export TRUST_DOCKER_IMAGE=1
​
b) export SIGN_DOCKER_IMAGE=1
​
c) export TRUST_DOCKER_CONTENT=1
​
d) export DOCKER_CONTENT_TRUST=1









































'D'

For the Docker client to know to sign a image to a trusted repository, you will need to set an environment variable pointing to it:

$ export DOCKER_CONTENT_TRUST=1
