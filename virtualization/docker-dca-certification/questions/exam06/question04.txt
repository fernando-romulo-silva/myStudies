
What is the possible option for running the multiple copies of a composed file in one host?

a) You need to set a custom project name by using the command "docker compose -p(--project-name) COMPOSE_PROJECT_NAME"
​
b) A custom project name is set by using the command "COMPOSE_PROJECT_NAME"
​
c) Both of the above
​
d) None of the above








































Answer 'c'

For running the multiple copies of a composed file on one host two options are available

1. You need to set a custom project name by using the command:

$ docker compose -p(--project-name) COMPOSE_PROJECT_NAME


2. A custom project name is set by using the command "COMPOSE_PROJECT_NAME"
