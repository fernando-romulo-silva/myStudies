Which of the following is required to install Docker EE from a package repository?

A)
License key obtained from Docker Hub
​
B)
Repository URL obtained from Docker Hub
​
C)
Repository URL obtained from Docker Store
​
D)
License key obtained from Docker Store











































C is correct

Install with a package:

1* Go to the Docker EE repository URL associated with your trial or subscription in your browser. 
2* Install Docker EE, changing the path below to the path where you downloaded the Docker package.
3* Start Docker.
4* Verify that Docker EE is installed correctly by running the hello-world image.
