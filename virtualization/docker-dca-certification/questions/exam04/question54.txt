Docker Hub and Docker Cloud are examples of?

A)
docker templates

B)
docker repos

C)
docker libraries

D)
docker registries










































Answer 'D'

Docker Hub is a cloud-hosted version of Docker Registry. 
A Docker user can opt for Docker Registry, which is a stateless, open source and scalable server-side application, 
if they prefer to maintain the storage and distribution of Docker images instead of relying on Docker's service.
