Which of the following option is used to specify a repository and a 
tag for saving the image after the build is done?
​
A)
-f
​
B)
-m
​
C)
-t
​
D)
-p




































C is correct

$ docker tag <Image-ID> <desired Name:Tag>

To specify a value for repository and tag of the image, we use the --tag or -t flag with the $ docker build command. 

The format of the value for this flag is <repository>:<tag>. 
If we leave the :<tag> part out, Docker will use the latest value for the <tag> part implicitly.
