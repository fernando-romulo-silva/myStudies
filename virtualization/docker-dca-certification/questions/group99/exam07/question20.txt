
Which of the following steps are required to create a Swarm backup?

a) Stop Docker Engine on any manager to ensure files are static.

b) Copy the /var/lib/docker/swarm directory content for backing up Swarm.

c) Raft logs should be backed up apart from normal files.

d) All of the preceding steps are required.









































Answer 'a' and 'b'

Performing back up and restoration on same server

1 stop the docker daemon

2 copy the /var/lib/docker/swarm folder

You don't neet backup the logs to backup the swarm
