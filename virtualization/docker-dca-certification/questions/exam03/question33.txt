The output of which command can be used to find the architecture 
and operating system an image is compatible with?
​
A)
docker image ls
​
B)
docker image inspect --filter '{{.Architecture}} {{.OS}}'
​
C)
docker image inspect --format '{{ .Architecture }} {{ .OS }}'
​
D)
docker image info




































C is correct

Docker inspect provides detailed information on constructs controlled by Docker.

By default, docker inspect will render results in a JSON array.

--format , -f		Format the output using the given Go template

$ docker image inspect --format '{{ .Architecture }} {{ .Os }}' mysql:5.7
amd64 linux
