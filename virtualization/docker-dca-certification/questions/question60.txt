
Which resources are used to link pods with Kubernetes' defined volumes?


a) persistentVolume

b) persistentVolumeClaim

c) storageClass

d) persistentDataVolume







































Answer(s) 'b'

Kubernetes persistent volume (PV) framework to allow cluster administrators to provision persistent storage for a cluster. 
Developers can use persistent volume claims (PVCs) to request PV resources without having specific knowledge of the underlying storage infrastructure.
