Which of the following command is used for checking the docker client and docker server?

​A)
$ docker version
​
B)
$ docker info
​
C)
$ docker login
​
D)
$ docker system prune





































A is Correct

$ docker system prune is used for is used removing all the stock containers.
 
$ docker login is used for promoting user name or password inserting those and finally, you are logged in.

$ docker info is used for getting the data information about the docker that is installed in your system.
Information displayed includes the kernel version, storage driver, number of containers and images. 
The number of images shown is the number of unique images.
