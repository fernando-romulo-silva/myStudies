Which command is used to remove a user-defined bridge network?

a) ​docker network 'network name' rm
​
b) docker network rm 'network name'
​
c) docker delete network 'network name'
​
d) docker-network rm 'network name'














































Answer 'B'

Removes one or more networks by name or identifier. 
To remove a network, you must first disconnect any containers connected to it.

$ docker network rm my-network
