
Which file is used to configure the default logging driver?

A)
/opt/docker/daemon.json
​
B)
/var/docker/daemon.json
​
C)
/etc/docker/daemon.json
​
D)
/etc/docker/startup.json










































Answer 'C'

To configure the Docker daemon to default to a specific logging driver, set the value of log-driver to the name of the logging driver in the daemon.json :

{
  "log-driver": "json-file",
  "log-opts": {
    "max-size": "10m",
    "max-file": "3",
    "labels": "production_status",
    "env": "os,customer"
  }
}
