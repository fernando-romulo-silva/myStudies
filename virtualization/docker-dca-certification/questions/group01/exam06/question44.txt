
You can create a volume dynamically using which command?

A)
docker create volume
​
B)
docker make volume
​
C)
Volumes cannot be created after container creation
​
D)
docker volume create










































Answer 'D'

Creates a new volume that containers can consume and store data in. 
If a name is not specified, Docker generates a random name.

$ docker volume create [OPTIONS] [VOLUME]

Ex:

$ docker volume create hello

Using the volume:

$ docker run -d -v hello:/world busybox ls /world
