
Which of the following commands will automatically create a volume when a container is started?

a) ​docker container run --name nginxtest --volumes myvol:/app:new nginx
​
b) docker container run --name nginxtest -v myvol:/app nginx
​
c) docker container run --name nginxtest -v /app:mount nginx
​
d) docker container run --name nginxtest --volumes=/app nginx








































Answer 'c'

The folder /app must be create before...

The -v (or --volume) argument to docker run is for creating storage space inside a container that is separate from the rest of the container filesystem.

* When given a single argument, like -v /var/lib/mysql, this allocates space from Docker and mounts it at the given location. 
 
* When given two arguments (host_path:container_path), like -v /data/mysql:/var/lib/mysql, this mounts the specified directory 
  on the host inside the container at the specified path
