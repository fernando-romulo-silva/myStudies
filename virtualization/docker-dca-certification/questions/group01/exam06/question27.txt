
Which of the following is required to install Docker EE from a package repository?

a) License key obtained from Docker Store

b) Repository URL obtained from Docker Hub
​
c) License key obtained from Docker Hub
​
d) Repository URL obtained from Docker Store








































Answer 'd'

Install with a package:

1 - Go to the Docker EE repository URL associated with your trial or subscription in your browser.
Go to centos/7/x86_64/stable-<VERSION>/Packages and download the .rpm file for the Docker version you want to install.

2 - Install Docker EE, changing the path below to the path where you downloaded the Docker package. ...
$ sudo yum install /path/to/package.rpm

3 - Start Docker
$ sudo systemctl start docker

4- Verify that Docker EE is installed correctly by running the hello-world image.
$ sudo docker run hello-world
