
A user is having problems running Docker. 
Which of the following will start Docker in debug mode?

a) Start the 'dockerd' process manually with the '--raw-logs' flag set to debug
​
b) Set the debug key to true in the 'daemon.json' file.

c) Start the 'dockerd' process manually with the '--logging' flag set to debug
​
d) Set the logging key to debug in the 'daemon.json' file.








































Answer 'b'

1 - Add param "-D" or "--debug"  to enable debug mode

Or

2 - You can create the file at /etc/docker/daemon.json if it does not already exist.

Add the following code snippet:

{
    "debug": true
}
