From the below-mentioned option which is the false advantage of Docker ?

​A)
Docker provides a difficult set-up initially.
​
B)
The application lifecycle can be described in detail, docker grants permission for it.
​
C)
Simple configuration and interaction with docker composed as possible.
​
D)
All of the above





































A is Correct

All the other states except A, are included in the advantages of the docker container. 

But the first statement is incorrect because the docker container provides an easy set-up which is quite efficient initially. 

Hence it is the correct answer.
