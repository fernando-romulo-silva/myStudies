By default manager nodes also run services as worker nodes, 
but you can configure them to run manager tasks exclusively and be manager-only nodes.

A)
TRUE

B)
FALSE











































The correct answer is A

By default manager nodes also run services as worker nodes, 
but you can configure them to run manager tasks exclusively and be manager-only nodes. 

An agent runs on each worker node and reports on the tasks assigned to it.
