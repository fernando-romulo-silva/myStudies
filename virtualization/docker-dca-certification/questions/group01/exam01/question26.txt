Which set of commands can identify the published port(s) for a container?

​A)
'docker container inspect', docker port'
​
B)
'docker network inspect', 'docker port'
​
C)
'docker port inspect', 'docker container inspect'
​
C)
'docker info', 'docker network inspect'





































The correct answer is A

$ docker port murachs-mysql

or 

$ docker inspect --format='{{.NetworkSettings.Ports}}' murachs-mysql
