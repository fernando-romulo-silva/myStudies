Which of the following command is used for stopping a running container?

​A) 
$ docker kill <container_id>

B)
$ docker stop <container_id>
​
C)
$ docker rm <container_id>
​
D)
$ docker start <container_id>





































B is Correct

$ docker kill <container_id> this command is used for killing a container. 
$ docker start <container_id> command is used for starting a docker container. 
$ docker rm <container_id> command is used for selecting a stopped container. 

Hence the correct option is B, which is used for stopping a running container.
