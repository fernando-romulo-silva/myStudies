Which of the following command is used for getting the detail information about the docker that is installed in your system?

​A)
$ docker version
​
B)
$ docker info
​
C)
$ docker login
​
D)
$ docker system prune





































B is Correct

The 'docker system prune' is used for is used removing all the stock containers. 

The 'docker login' is used for promoting user name or password inserting those and finally, you are logged in. 

For checking the docker client and docker server version the command used is 'docker version'.
